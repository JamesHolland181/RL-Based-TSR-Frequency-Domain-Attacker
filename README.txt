# TSR Workflow Package

## Contents
- TSR_Models/: CNN-3SPT, SAG-ViT, ResNet50, ViT model architectures
- Model_Weights/: Pre-trained model weights (.pth files)
- Adversarial_Training/: Scripts for adversarial training

## Models Included
1. CNN-3SPT (48x48 input, 671 LISA classes)
2. SAG-ViT (224x224 input, spatial-aware graph ViT)
3. ResNet50 (224x224 input)
4. ViT-B/16 (224x224 input)

## Usage
See individual Python files for model architectures and training scripts.
